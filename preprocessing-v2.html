  1. Pembuatan Supervised Data dengan Lag Features Mengubah data time series menjadi format supervised learning dengan membuat lag features (nilai-nilai pada hari sebelumnya) dan menambahkan fitur temporal seperti year, month, day, dan representasi siklik.

Fungsi Create Supervised Data Fungsi ini membuat lag features dengan menggeser nilai target sebanyak n_lags hari ke belakang, serta menambahkan fitur temporal:

Lag features: Nilai NO2 pada hari-hari sebelumnya

Fitur temporal: year, month, day, dayofweek, dayofyear

Fitur siklik: sin dan cos untuk month dan day (menangkap sifat siklik waktu)

def create_supervised_data(data, n_lags=7, target_col='NO2_mean'):
    df_supervised = data.copy()
    
    df_supervised = df_supervised.sort_values('date').reset_index(drop=True)
    
    print(f"Membuat {n_lags} lag features untuk prediksi...")
    
    for i in range(1, n_lags + 1):
        df_supervised[f'{target_col}_lag_{i}'] = df_supervised[target_col].shift(i)
    
    df_supervised['year'] = df_supervised['date'].dt.year
    df_supervised['month'] = df_supervised['date'].dt.month
    df_supervised['day'] = df_supervised['date'].dt.day
    df_supervised['dayofweek'] = df_supervised['date'].dt.dayofweek
    df_supervised['dayofyear'] = df_supervised['date'].dt.dayofyear
    
    df_supervised['month_sin'] = np.sin(2 * np.pi * df_supervised['month'] / 12)
    df_supervised['month_cos'] = np.cos(2 * np.pi * df_supervised['month'] / 12)
    df_supervised['day_sin'] = np.sin(2 * np.pi * df_supervised['day'] / 31)
    df_supervised['day_cos'] = np.cos(2 * np.pi * df_supervised['day'] / 31)
    
    df_supervised = df_supervised.dropna().reset_index(drop=True)
    
    return df_supervised
